# Pet_project_EDA_Netflix_dataset

## Описание проекта
В данном проекте был проведён анализ датасета Netflix Movies and TV Shows.
Цель проекта — провести очистку данных, исследовательский анализ (EDA),
визуализировать данные и построить аналитический дашборд.

Проект включает полный цикл анализа данных:
очистка данных → исследовательский анализ → визуализация → дашборд → выводы.

---

## Датасет
Использовался датасет Netflix Movies and TV Shows, содержащий информацию о фильмах и сериалах:
- тип контента
- страна
- дата добавления
- год выпуска
- рейтинг
- длительность
- жанры

---

## Очистка и подготовка данных
В процессе предобработки данных были выполнены следующие шаги:
- обработаны пропущенные значения
- удалены дубликаты
- преобразован столбец date_added в формат datetime
- созданы новые признаки: year_added и month_added
- разделены страны и жанры (explode)
- данные подготовлены для анализа и визуализации

---

## Исследовательский анализ данных (EDA)
В рамках анализа были исследованы:
- соотношение фильмов и сериалов
- количество контента по годам
- количество контента по странам
- топ-10 жанров
- распределение рейтингов
- количество контента по месяцам

---

## Dashboard
На основе подготовленных данных был создан дашборд в DataLens,
который позволяет анализировать:
- общее количество проектов
- количество стран
- количество жанров
- фильмы vs сериалы
- количество контента по годам
- топ стран
- топ жанров
- рейтинги
- фильтрацию по году, стране и типу контента

Ссылка на Dashboard: https://datalens.yandex/nsodi76c0biy7

## Основные выводы (Инсайты)
По результатам анализа можно сделать следующие выводы:

1. На Netflix значительно больше фильмов, чем сериалов.
2. После 2016 года наблюдается резкий рост количества контента на платформе. Данный рост, вероятно, связан с глобальным расширением Netflix в 2016 году, когда сервис стал доступен во многих странах мира, а также с увеличением инвестиций в производство оригинального контента (Netflix Originals).
4. Пик добавления контента пришёлся примерно на 2019 год. Снижение после 2020 может быть связано с пандемией COVID-19 и остановкой производства фильмов и сериалов.
5. Больше всего контента производится в США, затем в Индии и Великобритании.
6. Самые популярные жанры — International Movies, Dramas и Comedies.
7. Большинство контента имеет рейтинг TV-MA и TV-14, что говорит о
   направленности платформы преимущественно на взрослую аудиторию.
8. Netflix имеет очень разнообразную библиотеку контента,
   включающую фильмы и сериалы из большого количества стран и жанров.
